# 🎧 OATZA Audio Tool (Project008)

This is a Python-driven audio player interface for Terminal. 
It uses `yt-dlp` and `mpv` to stream audio for research and CLI development study.

## ⚠️ Disclaimer
- This project is for **Educational Purposes** only.
- No copyrighted material is hosted here.
- It is a tool for developers to study streaming APIs.

## 🛠 Setup (Termux)
1. pkg update && pkg upgrade -y
2. pkg install python ffmpeg mpv -y
3. pip install yt-dlp
4. termux-setup-storage
5. python oatza001
